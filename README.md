### 目的

这本书的目的是教你如何做组织一个机器学习项目所需的大量的决定。 你将学习：

* 如何建立你的开发和测试集

* 基本错误分析

* 如何使用偏差和方差来决定该做什么

* 学习曲线

* 将学习算法与人类水平的表现进行比较

* 调试推理算法

* 什么时候应该和不应该使用端到端的深度学习

* 按部进行错误分析
